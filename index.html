✦ KI Studio — Hilfe

🚀 Schnellstart

  • Prompt eingeben — Beschreibe dein Bild. Englisch für Stilbegriffe empfohlen, Deutsch funktioniert auch.
  • 📋 Vorlagen — 27 Prompts in 5 Kategorien. Klicken → einfügen → Tab springt zum nächsten [Platzhalter]
  • ✨ — Prompt automatisch mit KI verbessern und ausformulieren
  • ✕ — Prompt-Feld leeren
  • ⚡ Generieren klicken oder Strg+Enter
  • Ergebnis anklicken → Vollbild · ↓ PNG herunterladen · 📎 Als Eingabe weiterverwenden

🤖 Modelle & Kosten

  • Gemini 2.5 Flash — ~0,04 €/Bild · nimmt Bilder als Eingabe · Chat-Modus „Aufbauen" · Bild + Text als Ausgabe
  • Gemini 3 Pro — ~0,13 €/Bild · höchste Qualität · langsamer
  • Imagen 4 Fast — ~0,02 €/Bild · schnell · bis 4 Bilder gleichzeitig · nur Text → Bild
  • Imagen 4 Standard / Ultra — ~0,04–0,06 € · höhere Qualität · 2K wählbar
  • Imagen akzeptiert keine Bild-Eingabe — ausschließlich Text → Bild

⚙️ Parameter erklärt

  • Ausgabe: Bild & Text — Gemini liefert das Bild + einen kurzen Kommentar zur Interpretation. Hilfreich für Feedback.
  • Ausgabe: Nur Bilder — nur das Bild, kein Text. Schneller, kompakter.
  • Temperatur 0–2 — 0 = präzise & deterministisch · 1 = ausgewogen (Standard) · 2 = maximal kreativ & zufällig
  • Thinking-Level — lässt Gemini länger „nachdenken" vor dem Zeichnen. Mehr = bessere Qualität, aber langsamer & teurer. PIN-geschützt.
  • System-Kontext — unsichtbarer Hintergrundprompt: „Aus" = freie Hand · „Expert" = photorealistisch, geometrisch präzise · „Graustufen" = Schwarz-Weiß
  • Auflösung — Gemini: aktuell nur 1K (~1024 px); 2K/4K in Vorbereitung. Imagen Standard/Ultra: 1K oder 2K wählbar. Imagen Fast: immer 1K.

🎲 Seeds

  • Jedes Bild zeigt unten ein 🎲 Seed-Tag mit der verwendeten Zufallszahl
  • Seed notieren → ins Seed-Feld eingeben → gleicher Prompt = sehr ähnliches Ergebnis reproduzierbar
  • 🎲 Button — zufälligen Seed generieren · Feld leer lassen = komplett zufällig
  • Seeds funktionieren zuverlässig bei gleichem Modell, Prompt und Einstellungen

📎 Bilder als Eingabe

  • Bis zu 5 Bilder hochladen — Klick auf „📎 Bilder" oder Drag & Drop auf das Textfeld
  • Gemini nutzt die Bilder als visuelle Referenz: Stil übernehmen, Materialien tauschen, auf einem Foto aufbauen
  • „📎 Als Eingabe" — erscheint beim Überfahren eines generierten Bildes. Lädt dieses Bild direkt als Eingabe für die nächste Generierung. Kein Download, kein erneutes Hochladen nötig — ideal für schrittweise Überarbeitungen.
  • Einzelne Bilder mit ✕ entfernen · alle mit „Alle ✕"
  • Imagen-Modelle unterstützen keine Bild-Eingabe

↺ Aufbauen (Chat-Modus)

  • Nur für Gemini-Modelle verfügbar
  • „Neue Serie" — jede Generierung startet unabhängig, kein Kontext aus vorherigen Bildern
  • „Aufbauen" — Gemini erinnert sich an vorherige Generierungen und verfeinert schrittweise. Ideal für iterative Überarbeitungen.
  • Beim Modellwechsel wird der Chat-Verlauf automatisch zurückgesetzt

💾 Verlauf & Speichern

  • Bilder werden lokal im Browser gespeichert — bleiben nach Tab-Schließen erhalten
  • Max. 8 Bilder im Verlauf; älteste werden automatisch entfernt
  • ↓ PNG — Einzelbild speichern · Alle PNG — alle Bilder auf einmal herunterladen
  • „Verlauf löschen" entfernt alle gespeicherten Bilder dauerhaft

💡 Höhere Auflösung

Gemini und Imagen Fast liefern ~1024 px. Für Druckqualität: PNG in Photoshop öffnen → Bearbeiten → Generative Upscale (Adobe Firefly) → bis 4× Vergrößerung ohne sichtbaren Qualitätsverlust.

🚀 Quick Start

  • Enter a prompt — Describe your image. English recommended for style terms, German works too.
  • 📋 Templates — 27 prompts in 5 categories. Click → insert → Tab jumps to next [placeholder]
  • ✨ — Auto-improve and expand the prompt using AI
  • ✕ — Clear the prompt field
  • Click ⚡ Generate or press Ctrl+Enter
  • Click result → full screen · ↓ PNG download · 📎 As input to reuse

🤖 Models & Costs

  • Gemini 2.5 Flash — ~€0.04/image · accepts image input · chat mode "Build on" · Image + Text output
  • Gemini 3 Pro — ~€0.13/image · highest quality · slower
  • Imagen 4 Fast — ~€0.02/image · fast · up to 4 images at once · text to image only
  • Imagen 4 Standard / Ultra — ~€0.04–0.06 · higher quality · 2K available
  • Imagen does not accept image input — text to image only

⚙️ Parameters explained

  • Output: Image & Text — Gemini returns the image + a short comment. Useful for feedback and explanations.
  • Output: Image only — just the image, no text. Faster, less distraction.
  • Temperature 0–2 — 0 = precise & deterministic · 1 = balanced (default) · 2 = maximally creative & random
  • Thinking Level — lets Gemini "think" longer before drawing. More = better quality, slower & more expensive. PIN-protected.
  • System Context — invisible background prompt: "Off" = free rein · "Expert" = photorealistic, geometrically precise · "Grayscale" = black & white only
  • Resolution — Gemini: currently only 1K (~1024 px); 2K/4K coming soon. Imagen Standard/Ultra: 1K or 2K. Imagen Fast: always 1K.

🎲 Seeds

  • Every image shows a 🎲 seed tag at the bottom with the number used
  • Note the seed → enter in seed field → same prompt = very similar result reproducible
  • 🎲 Button — generate a random seed · leave field empty = fully random
  • Seeds work reliably with the same model, prompt, and settings

📎 Images as input

  • Upload up to 5 images — click "📎 Images" or drag & drop onto the text field
  • Gemini uses images as visual reference: copy a style, swap materials, build on a photo
  • "📎 As input" — appears when hovering a generated image. Loads it directly as input for the next generation. No download needed — ideal for step-by-step revisions.
  • Remove individual images with ✕ · remove all with "All ✕"
  • Imagen models do not support image input

↺ Build on (Chat mode)

  • Only available for Gemini models
  • "New series" — each generation starts independently, no context from previous images
  • "Build on" — Gemini remembers previous generations and refines step by step. Ideal for iterative revisions.
  • Chat history is automatically reset when switching models

💾 History & Saving

  • Images are stored locally in the browser — they persist after closing the tab
  • Max. 8 images in history; oldest are automatically removed
  • ↓ PNG — save individual image · All PNG — download all images at once
  • "Clear history" permanently removes all stored images

💡 Higher Resolution

Gemini and Imagen Fast output ~1024 px. For print quality: open PNG in Photoshop → Edit → Generative Upscale (Adobe Firefly) → up to 4× enlargement without visible quality loss.
